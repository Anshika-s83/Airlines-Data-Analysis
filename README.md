# ✈️ Airlines Data Analysis Project

## Overview
This project analyzes airline operations data to uncover trends, delays, cancellations, airline performance, and destination insights. The analysis is performed using Python, SQL, and data visualization techniques to generate meaningful business insights from flight data.

## Objectives
The project focuses on the following key performance indicators (KPIs):

- Monthly flight trends
- Top airlines by number of flights
- Average arrival delay by airline
- Cancellation analysis
- Delay distribution analysis
- Departure delay vs arrival delay relationship
- Delay heatmap by hour and month
- Top destinations analysis
- Flight distance vs air time analysis
- Interactive dashboard for complete dataset

## Tech Stack

- Python
- Pandas
- NumPy
- MySQL
- SQLAlchemy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
The project uses airline operational data stored in a MySQL database, including:

- Flight information
- Airport details
- Airline details
- Departure and arrival times
- Delay information
- Cancellation records
- Distance and airtime metrics

## Project Workflow

1. Connect to MySQL database
2. Extract airline datasets
3. Perform data cleaning and preprocessing
4. Handle missing values and cancelled flights
5. Conduct exploratory data analysis (EDA)
6. Generate visualizations and insights
7. Build an interactive dashboard

## Key Insights Generated

### Monthly Flight Trends
Analyze how flight volumes change across different months.

### Airline Performance
Compare airlines based on:
- Total flights operated
- Average arrival delays
- Cancellation rates

### Delay Analysis
Study:
- Departure delays
- Arrival delays
- Delay distributions
- Correlation between departure and arrival delays

### Destination Analysis
Identify the busiest destinations and travel patterns.

### Operational Efficiency
Evaluate the relationship between flight distance and air time.
