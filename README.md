📌 Project Overview

This project analyzes flight performance data from 2013 to understand airline delays, cancellations, and operational efficiency. The goal is to extract insights that help evaluate airline reliability and identify patterns in flight delays.

🛠️ Tools & Technologies
Python
Pandas
NumPy
Matplotlib
Seaborn
SQL (MySQL)
Streamlit
📂 Dataset

The dataset contains information about flights including:

Departure & arrival times
Delay durations
Airlines and routes
Cancellations
Distance and air time
🎯 Key Objectives
Identify airlines with highest and lowest delays
Analyze cancellation rates across carriers
Study delay patterns over time
Explore relationship between departure and arrival delays
Visualize flight trends and distributions
📊 Key Insights
1. Airline Performance

Some airlines consistently show higher average delays compared to others.

2. Delay Propagation

Strong correlation exists between departure delay and arrival delay.

3. Cancellations

Certain carriers have significantly higher cancellation rates.

4. Time Trends

Flight volume and delays vary across months and hours of the day.
